# Quickly get started with a typescript project.

![Create a typescript library](/ct-l-4.gif)

## Use without installing globally

```sh
npx create-ts
```

## Install globally

```sh
npm i -g create-ts
```

## Create a typescript library

```sh
npx create-ts -l
```

## Create a typescript react component

```sh
npx create-ts -rc
```
